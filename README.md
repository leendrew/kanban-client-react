# Simple Kanban Dashboard (Client)

[Server Repo](https://github.com/leendrew/kanban-server-nest)

[Deployed](https://kanban-client-react.onrender.com)

## Stack

- React
- Redux Toolkit

## Start

### dev

```bash
cp .env.example .env
yarn
yarn dev
```

### prod

```bash
cp .env.example .env
yarn
yarn build && yarn preview
```
